# EduNexus

EduNexus это многостраничный фронтенд проект образовательной платформы для учителей учеников и администраторов

## Что есть в проекте

- `index.html` главная страница и форма регистрации
- `teachers.html` кабинет учителя
- `students.html` кабинет ученика
- `admin.html` админ панель для загрузки материалов
- `styles.css` единая визуальная система
- `script.js` логика регистрации загрузки и просмотра материалов

## Функционал

- отдельные страницы для ролей
- регистрация с проверкой полей
- публикация материалов через админ панель
- просмотр материалов и презентаций прямо на сайте
- хранение данных пользователей и библиотеки в localStorage

## Запуск

```bash
python3 -m http.server 8000
```

Откройте в браузере

- `http://127.0.0.1:8000/index.html`
- `http://127.0.0.1:8000/teachers.html`
- `http://127.0.0.1:8000/students.html`
- `http://127.0.0.1:8000/admin.html`
