# EduNexus

EduNexus — современный многостраничный front-end проект образовательной платформы для учителей, учеников и администраторов.

## Что изменилось

Теперь это не одна landing-page, а набор отдельных страниц по ролям:

- `index.html` — главная страница и навигационный хаб платформы;
- `teachers.html` — кабинет учителя;
- `students.html` — кабинет ученика;
- `admin.html` — админ-панель с загрузкой материалов;
- `styles.css` — общая визуальная система;
- `script.js` — логика библиотеки материалов, рендера каталогов и загрузки через localStorage.

## Функционал

- multi-page структура вместо одного экрана;
- отдельные сценарии для учителей, учеников и админов;
- загрузка учебных материалов в админке;
- автоматическое отображение загруженных материалов на страницах ролей;
- хранение каталога в `localStorage` без backend;
- адаптивный modernist UI.

## Запуск

Запустите статический сервер в корне проекта:

```bash
python3 -m http.server 8000
```

После этого откройте:

- `http://127.0.0.1:8000/index.html`
- `http://127.0.0.1:8000/teachers.html`
- `http://127.0.0.1:8000/students.html`
- `http://127.0.0.1:8000/admin.html`
